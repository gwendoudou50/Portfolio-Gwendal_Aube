Jeu de Yams

Projet IUT

Le Yams est un projet de réalisation d'un jeu de dés pour 2 jours. J'ai d'abord réalisé une maquette de l'affichage sur la console. Puis j'ai réalisé un cahier de tests pour ensuite écrire le programme principal en pseudo-code. Enfin j'ai développé le jeu en langage C en incluant des commentaires formaté dans le code afin de pouvoir créer une documentation à l'aide de Doxygen. Les difficultés du projets étaient d'appréhender la documentation Doxygen et de bien réfléchir à toutes les fonctionnalités et possibilités de bug lors de la rédaction du cahier de tests et du pseudo-code.
